[공통 로직]  
(1) 첫 어휘력 측정 전 이탈 알림, 문법 학습 알림은 맞춤형 알림 A 발송 시간에 발송

(2) 당일 알림 발송 전 하나라도 트로피 획득한 경우 발송하지 않음

(3) 맞춤형 알림 A 미접속 10일차 “이제 알림은 그만 보낼게요..” 발송 이후에는 모든 학습 리마인드 알림 발송하지 않음

[맞춤형 알림 A(복습 문장 알림) 발송 로직]  
(1) 평일에는 평일 학습 기록을 기준으로 발송  
(1-1) 평일 학습 기록이 없는 경우(주말 학습 기록만 존재)  
직전 주말 학습일의 첫 학습 시작 시간에 발송  
(1-2) 평일 학습 기록이 1일 이상 ~ 3일 미만인 경우  
직전 평일 학습일의 첫 학습 시작 시간에 발송  
       ex) 오전 11시/오후 3시에 학습을 시작한 경우 오전 11시에 발송  
(1-3) 평일 학습 기록이 3일 이상 있는 경우  
최근 평일 3일 동안의 첫 학습 시작 시간의 중앙값 계산 후,  
3개의 시간이 모두 중앙값에서 +-30분 내에 있다면 중앙값에 발송  
중앙값 내에 없다면 직전 평일 학습일의 첫 학습 시작 시간에 발송  
  
(2) 주말에는 주말 학습 기록을 기준으로 발송  
(2-1) 주말 학습 기록이 없는 경우(평일 학습 기록만 존재)  
직전 평일 학습일의 첫 학습 시작 시간에 발송  
(2-2) 주말 학습 기록이 1일 이상 ~ 3일 미만인 경우  
직전 주말 학습일의 첫 학습 시작 시간에 발송  
(2-3) 주말 학습 기록이 3일 이상 있는 경우  
최근 주말 3일 동안의 첫 학습 시작 시간의 중앙값 계산 후,  
3개의 시간이 모두 중앙값에서 +-30분 내에 있다면 중앙값에 발송  
중앙값 내에 없다면 직전 주말 학습일의 첫 학습 시작 시간에 발송  
  
(공통)

맞춤형 알림 A의 평일 발송 예정 시간이 10:20 전인 경우, 발송하지 않음  
  
(2) 맞춤형 알림 A의 주말 발송 예정 시간이 12:20 이후인 경우,  주말 10:00 ~ 10:30 랜덤 발송
